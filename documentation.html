VEDARA Documentation

Complete guide to using VEDARA's agricultural management platform.

Getting Started

Welcome to VEDARA documentation. This guide will help you understand how to use our platform to manage your farming operations efficiently.

Prerequisites: Before you begin, ensure you have a VEDARA account. If you don't have one, you can sign up for free.

Quick Setup

# Step 1: Create your account
1. Visit https://vedara.com/signup
2. Enter your farm details
3. Verify your email address

# Step 2: Add your first farm
1. Navigate to "My Farms"
2. Click "Add New Farm"
3. Enter farm location and details

# Step 3: Start tracking crops
1. Go to "My Crops"
2. Add your first crop
3. Set planting and harvest dates

Farm Management

VEDARA allows you to manage multiple farms from a single dashboard. Each farm can have its own properties, crops, and settings.

Adding a New Farm

POST
/api/v1/farms

Request Body:

{
  "name": "Green Valley Farm",
  "location": {
    "latitude": -1.2921,
    "longitude": 36.8219
  },
  "area_acres": 12.5,
  "soil_type": "volcanic_loam",
  "irrigation_type": "drip"
}

Farm Properties

Property Type Description Required
name String Farm name (max 100 characters) Yes
location Object GPS coordinates {latitude, longitude} Yes
area_acres Float Total farm area in acres Yes
soil_type String Type of soil (dropdown selection) No
irrigation_type String Irrigation system type No

Crop Tracking

Track crop growth stages, health, and harvest predictions with our comprehensive crop management system.

Tip: Regular updates (at least weekly) provide the most accurate growth predictions and health assessments.

Adding a Crop

// Example: Adding maize crop to farm ID 123
const cropData = {
  farm_id: 123,
  name: "Maize",
  variety: "DH04",
  planting_date: "2024-01-15",
  expected_harvest_date: "2024-05-30",
  area_acres: 2.5,
  status: "growing",
  notes: "Planted in section A with drip irrigation"
};

// API Request
fetch('/api/v1/crops', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify(cropData)
});

Weather Data Integration

VEDARA provides hyper-local weather data for your farm location, including forecasts, historical data, and alerts.

Getting Weather Data

GET
/api/v1/weather/{farm_id}/forecast?days=7

Response Example:

{
  "farm_id": 123,
  "location": {
    "name": "Green Valley Farm",
    "coordinates": {"lat": -1.2921, "lng": 36.8219}
  },
  "forecast": [
    {
      "date": "2024-01-20",
      "max_temp": 28,
      "min_temp": 16,
      "precipitation": 12,
      "condition": "partly_cloudy"
    }
  ],
  "alerts": []
}
Note: Weather data is updated hourly. For the most accurate agricultural recommendations, use our weather-based irrigation and planting suggestions.
