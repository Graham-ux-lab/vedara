Build powerful agricultural applications with our comprehensive REST API. Access farm data, weather information, crop analytics, and more programmatically.

API Features

Real-time Data Access

Access live farm data including crop health, weather conditions, soil moisture levels, and equipment status in real-time.

Comprehensive Endpoints

Over 50 RESTful endpoints covering farms, crops, weather, market data, analytics, and user management.

Webhook Support

Set up webhooks to receive notifications for events like harvest alerts, weather warnings, and price changes.

Rate Limiting

Generous rate limits with tier-based access. Enterprise plans offer unlimited API calls.

Data Export

Export farm data in JSON, CSV, or XML formats for integration with other systems and analytics tools.

Sandbox Environment

Test your integration with our full-featured sandbox environment before going to production.

Quick Start

GET

Get API Key

GET https://api.vedara.com/v1/auth/apikey

First, obtain your API key from the dashboard. You'll need this for all API requests.

GET

Test Connection

GET https://api.vedara.com/v1/ping

Test your connection to the VEDARA API:

curl -X GET \
-H "Authorization: Bearer YOUR_API_KEY" \
https://api.vedara.com/v1/ping

Response:

{
  "status": "success",
  "message": "API is working",
  "timestamp": "2024-01-15T10:30:00Z"
}

Authentication

1

Get Your API Key

Navigate to Settings ‚Üí API Access in your VEDARA dashboard to generate your API key.

2

Include in Requests

Include the API key in the Authorization header of all your requests:

Authorization: Bearer sk_live_1234567890abcdef
3

Test Authentication

Verify your authentication is working by making a test request:

# Test with curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
https://api.vedara.com/v1/farms

Example Endpoints

GET

List Farms

GET https://api.vedara.com/v1/farms

Retrieve a list of all farms associated with your account.

GET

Get Crop Details

GET https://api.vedara.com/v1/farms/{farm_id}/crops/{crop_id}

Retrieve detailed information about a specific crop.

POST

Create New Crop

POST https://api.vedara.com/v1/farms/{farm_id}/crops

Create a new crop record:

{
  "name": "Maize",
  "variety": "DH04",,
  "planting_date": "2024-01-15",
  "area_acres": 2.5,
  "status": "growing"
}
GET

Get Weather Data

GET https://api.vedara.com/v1/weather/{location_id}/forecast

Retrieve weather forecast for a specific farm location.
